## Guide vs How-To

**Development Guides** provide comprehensive understanding:

- Explain concepts in depth
- Show multiple approaches
- Discuss trade-offs and best practices
- Suitable for learning

**How-To Guides** provide task-specific instructions:

- Step-by-step procedures
- Get things done quickly
- Minimal explanation
- Suitable for reference

For quick tasks, see [How-To Guides](../howto/index.md).
