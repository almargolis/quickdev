There are a number of operational things that must be handled before your application
can run. There are a lot of ways to do take care of those prerequisites and this
application can adapt to almost anything approach you want to take.

This application is built using QuickDev. QuickDev includes tools for quickly
handling most DevOps needs in a consistent, documented way. See the QuickDev DevOps Guide
to learn what needs to be done and what tools QuickDev provides to help.

Things that need to be handled:

* Configure Server
    * QuickDev mainly supports Linux and MacOS
    * Install / Configure webserver
        * QuickDev mainly supports Apache configuration but runs fine under any server.
        * Configure virtual hosts if your server supports multiple sites
* DNS Domain
    * create domain
    * create host records
