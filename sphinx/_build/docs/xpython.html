XPython¶

XPython is a pre-processor that helps maximize production code consistency and maximize DRY (Don’t Repeat Yourself) implementation.

Xpython is a preprocessor for python that adds data modeling and structured programming features without interfering with normal python fundamentals.

Xpython was developed for CommerceNode but does not use any of it’s modules because it needs to run regardless of its state. It’s simplest to keep this stand-alone.
